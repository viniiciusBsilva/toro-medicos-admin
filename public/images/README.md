# Imagens do projeto Tôro Médicos Admin

Coloque aqui as imagens usadas no projeto.

## Imagem da tela de login (lado direito)

A imagem do celular que aparece à direita nas telas de login deve ser salva como:

- **`image_login.png`** (ou `image_login.jpg` / `image_login.webp`)

Ou, se preferir pasta com espaço no nome:

- **`image login/image_login.png`** (arquivo dentro da pasta `image login`)

O layout das páginas de login tenta carregar, nesta ordem:
`image_login.png`, `image_login.jpg`, `image_login.webp`, `image login.png`, `image login.jpg`.

## Outras imagens

- `logo.svg` ou `logo.png` — logotipo Tôro Médicos (header, etc.)
- No código, use o caminho a partir de `/images/`, por exemplo: `/images/logo.svg`.
